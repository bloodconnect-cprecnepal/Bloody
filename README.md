# Bloody

## Project Overview

Bloody is an innovative platform aiming to connect blood donors with those in need, enabling a seamless and efficient way of saving lives. The project utilizes modern web technologies to ensure accessibility and ease of use.

## Features
- **User Registration**: Easy sign-up process for donors and recipients.
- **Search Functionality**: Locate donors quickly based on blood type and location.
- **Profile Management**: Users can manage their profiles and donation history.
- **Notifications**: Alerts for upcoming donation events or requests for blood.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Cloud Storage**: AWS S3 for profile pictures and files

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/bloodconnect-cprecnepal/Bloody.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Bloody
   ```
3. Install dependencies for both client and server:
   ```bash
   npm install
   ```
4. Set up environment variables as required.
5. Run the application:
   ```bash
   npm start
   ```

## Usage
- Navigate to the app URL to start using the platform.
- Donors can fill out their profiles and start donating.
- Recipients can search for donors based on their needs.
