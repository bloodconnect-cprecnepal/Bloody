# Contributing Guidelines

## Getting Started
1. Fork repository
2. Create feature branch
3. Make changes
4. Submit pull request

## Code Standards
- Use ES6+ syntax
- Follow ESLint rules
- Add JSDoc comments
- Max line length: 100 chars

## Testing
- Write tests for features
- Maintain 80%+ coverage
- Run: npm test

## Commit Convention
- feat: New feature
- fix: Bug fix
- docs: Documentation
- test: Tests
- style: Formatting
- refactor: Code reorganization

**Thank you for contributing! 🎉**
